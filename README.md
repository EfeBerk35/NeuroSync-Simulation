# NeuroSync-Simulation 
Proje Adı: NeuroSync - BCI Interface Simulator
Proje Hakkında: NeuroSync, Python kullanılarak geliştirilmiş, kurgusal bir Beyin-Bilgisayar Arayüzü (Brain-Computer Interface) simülasyonudur. Bu uygulama, ham EEG verilerinin görselleştirilmesi, işlenmesi ve komuta dönüştürülmesi sürecindeki kullanıcı deneyimini (UX) sergiler.

Özellikler:
-Cyberpunk UI: Modern, karanlık tema ve neon renk paleti ile fütüristik tasarım.

-Prosedürel Animasyon: math.sin fonksiyonları kullanılarak oluşturulan, her seferinde benzersiz ve akıcı EEG dalga simülasyonu.

-Event-Driven Akış: Tkinter'ın .after() metodu ile yönetilen, kare kare (frame-by-frame) işleyen 14 saniyelik senaryo motoru.

-Reaktif Arayüz: İşlem sırasına göre (Veri Alımı -> İşleme -> Çıktı) panellerin dinamik olarak vurgulanması (Highlighting).

-Kullanılan Teknolojiler:

-Python 3.x

-Tkinter (Standart GUI Kütüphanesi)
